**Customer Behavior Analysis**
End-to-End Data Analytics Project | Python • SQL • Power BI

 **Project Overview**
This project analyzes customer transaction data to uncover purchase patterns, customer value, loyalty trends, and revenue drivers. The goal is to transform raw data into actionable business insights using Python, SQL, and Power BI — similar to real-world analytics workflows.

** Objectives**
Understand customer purchasing behavior
Identify high-value and loyal customers
Analyze order frequency and revenue trends
Build an interactive dashboard for stakeholders

**Tools**
Python – Data cleaning & exploratory analysis
SQL – KPI extraction and aggregations
Power BI – Data modeling and visualization
Excel – Data validation and cross-checks

**** Project Workflow**
1️⃣ Data Cleaning (Python)
Removed duplicates and handled missing values
Standardized customer and transaction data
Ensured data quality and consistency

2️⃣ Exploratory Data Analysis
Customer distribution and revenue trends
Purchase frequency analysis
Identification of behavioral patterns

3️⃣ SQL Analysis
Total customers and orders
Average order value
Repeat and loyal customer identification
Customer segmentation using aggregations

4️⃣ Power BI Dashboard
KPI cards (Customers, Orders, Revenue)
Customer segmentation visuals
Trend analysis over time
Interactive filters for drill-down insights

**Key Insights**
A small percentage of customers contribute a large share of revenue
Loyal customers have significantly higher purchase frequency
Certain customer segments show churn risk
KPI-driven dashboards improve decision-making speed

**Business Impact**
Enables data-driven customer retention strategies
Helps identify and target high-value customers
Supports marketing and revenue optimization decisions
